

  ## ビルド

   BroccoliEngine  は動的ライブラリ (DLL) ではなく、静的ライブラリ ( .lib ) としてゲームプロジェクトに結合されます。


  ソリューションには以下のビルド構成が定義されています。

   構成名  | プラットフォーム | 役割                       | 定義
  ---------|------------------|----------------------------|---------------------------------------------------------
   Debug   | x64 / x86        | ゲームデバッグビルド       |  _DEBUG  
   Editor  | x64 / x86        | エディタ機能付き開発ビルド |  _EDITOR
   Release | x64 / x86        | 本番リリース用ビルド       |  _RELEASE
